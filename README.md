# Perpetuus

## Installation:

## Usage
1. Clone the `perpetuus` project. 
2. Inside the directory.
```sh
cargo build
```

## Development
```
cargo watch -x run
```

**Note:**
Add `cargo-watch` to auto build. 
```sh
cargo add cargo-watch
```
